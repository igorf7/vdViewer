Трек-линия

Трек-линия перемещается нажатием клавиш "Стрелка вправо" и "Стрелка влево". При этом в соответствующих полях вывода отображаются значения параметров в точках пересечения трек-линии с графиками. Клавишами "Home" и "End" можно установить трек-линию в начало или конец области просмотра. Если выбрано два параметра для просмотра графиков, в третьем поле вывода отображается их разность.


Зум

Можно выделить мышью какой-либо участок графика и он развернётся на всю область просмотра. Возврат к исходному состоянию - клик правой кнопкой мыши.





Прокрутка

При использовании зума изображение графиков можно прокручивать вверх - вниз - влево - вправо клавишами W A S D.

Меню Файл - Открыть

Открывает выбранный лог-файл для просмотра в виде графиков и таблицы.

Меню Файл - Закрыть

Закрывает текущий просмотр.

Меню Файл - Кодировка

Если вместо кириллицы отображаются зюкалки и крокозябры, следует выбрать другую кодировку в этом меню.

Меню Настройки - Показывать легенду

Позволяет включить/отключить показ легенды для графиков. Эта опция доступна после открытия файла для просмотра графиков.

Меню Настройки - Настройки лог-файла

Эти настройки по умолчанию позволяют корректно отображать большинство лог-файлов, но в иногда может возникнуть необходимость их изменить. Если после изменения настроек файл отображается не корректно, следует вернуться к настройкам по умолчанию. Для этого нужно в окне настроек установить флажок "Настройки по умолчанию" и нажать кнопку "Применить настройки". В большинстве случаев настройки по умолчанию являются оптимальными.
